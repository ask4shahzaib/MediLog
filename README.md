# MediLog

**To install the dependencies of this project you have to run this command *pip install -r install.txt* in terminal.**</br>
**The dependencies file is available in the project.**</br>
**Then to start the server you have to type *py manage.py runserver* in the terminal and the django server will start automatically.**</br></br>
